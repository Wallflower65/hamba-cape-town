# 🧭 Cape Town Traveling App (Hamba Kapa)

A dynamic travel routing web application designed specifically for navigating Cape Town. This project utilises the **OSRM (Open Source Routing Machine) API** to provide accurate geocoding and live route calculations, helping users find the most efficient paths across the Mother City. 

## ✨ Features

* **Live Routing:** Calculates optimal travel routes between points using high-performance routing data.
* **Geocoding Capabilities:** Seamlessly converts user-inputted Cape Town locations and addresses into exact geographic coordinates.
* **Interactive UI:** A clean, responsive frontend interface built for ease of use on the go.

## 🛠️ Tech Stack

* **Language:** JavaScript (ES6+)
* **Markup & Styling:** HTML5, CSS3
* **API Integration:** [OSRM API](http://project-osrm.org/) for routing and spatial data.
